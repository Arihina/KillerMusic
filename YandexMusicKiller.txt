Функциональные требования:

• Регистрация пользователя:
	o Создание, заполнение, запрос к БД
• Поиск трека / исполнителя по запросу
	o Запрос на поиск в БД
• Добавление своих треков на площадку 
	o Загрузка трека в хранилище
	o Добавление информации о треке в БД
• Удаление своих треков с площадки
	o Удаление трека из БД
	o Удаление трека из хранилища данных
• Создание личных плейлистов
	o Создание списка c ссылками на треки в хранилище
	o Привязка плейлиста к пользователю в БД
• Удаление личного плейлиста
	o Удаление плейлиста из БД
• Возможность добавления текста песни
	o Привязка текста к треку в БД


Use Case:

Регистрация -> Добавление пользователя в БД -> Предоставление прав пользования
Аутентификация -> Запрос в БД -> Предоставление прав пользования (если пользователь существует, иначе регистрация)

Добавление трека на площадку -> Загрузка трека в хранилище -> Добавление трека в БД -> Добавление трека в личный плейлист 
Добавление / изменение текста песни (для uploader) -> Запрос на добавления текста в БД -> Добавление текста в БД
Создание плейлиста (для creator) -> Запрос на добавление плейлиста в БД -> Привязка плейлиста к пользователю в БД -> Добавление треков в плейлист -> Указание приватности

Удаление трека с площадки (для uploader) -> Запрос на удаление трека в БД -> Удаление трека из плейлистов –> Удаление трека из хранилища
Удаление трека из плейлиста (для creator) -> Запрос на удаление трека из плейлиста в БД -> Удаление трека из списка треков в плейлисте
Удаление плейлиста (для creator) -> Запрос на удаление плейлиста из БД –> Удаление плейлиста у других пользователей (если плейлист не private)

Поиск трека по названию -> Получение названия трека от user -> Поиск в БД совпадений по полю name у SoundTracks -> Вывод списка треков с совпадающими значениями 
Поиск трека по исполнителю -> Получение имени исполнителя от user -> Поиск в БД совпадений по полю artist у SoundTracks -> Вывод списка треков с совпадающими значениями
Поиск трека по тексту -> Получение текста (или части текста) от user -> Поиск в БД частичных совпадений по полю Text у SoundTracks -> Вывод списка треков с совпадающими значениями
